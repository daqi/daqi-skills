# Daqi Skills

A collection of article writing and content creation skills

## Skills

- **article-analyzer** - Article analysis tool
- **article-outliner** - Article outline generator
- **article-polish** - Article polishing and refinement
- **article-producer** - Article production workflow
- **article-writer** - Article writing assistant
- **xhs-writer** - Xiaohongshu viral copywriting assistant
- **xhs-md2img** - Render XHS Markdown to fixed-size images
- **xhs-md-to-plain-text** - Convert Markdown to plain text for Xiaohongshu posting
- **html-parser-rule** - HTML parsing rules
- **skill-manager** - Skill management utility

## Installation

```bash
npx skills add daqi-skills
```
